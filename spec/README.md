# HyperMap Specification

The HyperMap specification, written in
[Bikeshed](https://speced.github.io/bikeshed/).

Read the spec at [hypermap.dev/spec](https://www.hypermap.dev/spec/).

## Building

Requires [Bikeshed](https://speced.github.io/bikeshed/) installed via `uv` or
`pipx`:

```sh
make
```

To watch for changes:

```sh
make watch
```
