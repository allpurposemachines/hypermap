# HyperMap Explorer

An interactive browser-based tool for exploring HyperMap resources. Enter a URL
and inspect the parsed HyperMap tree, navigate links, and submit controls.

Try it at [hypermap.dev/explorer](https://www.hypermap.dev/explorer/).

## Running locally

It's a single HTML file — just open `index.html` directly from `file://`.
